# SpaceShooter
Jeu d'arcade dans lequel le joueur doit éviter des astéroides
mais aussi tirer dessus pour marquer des points

## Setup

```bash
git clone https://github.com/EddyFrc/prog-moteur-tp1-space-shooter.git SpaceShooter
cd SpaceShooter
```

Puis ouvrir SpaceShooter.uproject avec Rider, Visual Studio ou autre... + Unreal Editor

## Installation

### Releases
Télécharger le zip correspondant à votre plateforme sur la
page [Releases](https://github.com/EddyFrc/prog-moteur-tp1-space-shooter/releases)

### Empaqueter depuis les sources
Voir Setup, puis utiliser l'option "Package project" depuis "Platforms" sur Unreal Editor
